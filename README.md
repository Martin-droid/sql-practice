# SQL Practice — Customers, Products & Orders

SQL exercises over a simple e-commerce schema (MySQL syntax).

## Files

- [schema.sql](schema.sql) — table definitions for `customers`, `products`, and `orders`
- [queries.sql](queries.sql) — the 10 practice queries:
  1. All data of customers
  2. `product_name` and `category` for products priced between 5000 and 10000
  3. All products sorted by price (descending)
  4. Total number of orders, average / highest / lowest total amount
  5. Number of orders per `product_id`
  6. Customers (`customer_id`) with more than 2 orders
  7. Number of orders per month in 2020
  8. Each order with its `product_name`, `customer_name`, and order date
  9. Orders made three months ago
  10. Customers who have never ordered a product
